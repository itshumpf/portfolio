title: "Higher-dimensional features"
category: "frontier"
order: 4
---
The current model uses two features (CFO, SFO) and discards the per-subcarrier phase residual
shape, which carries multipath and hardware-nonlinearity signature. Adding a residual-vector
feature to a higher-dimensional Mahalanobis model is the path past the ~77% same-model ceiling —
and it can be validated against the existing 2.8M-frame dataset with no new captures required.
